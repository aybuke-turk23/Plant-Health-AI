   ## 🌿  Plant-Health-AI
👋Hi! This is our software engineering project.We use Artificial Intelligence System to find diseases in plant leafs.

## 📝 About Project:
We use Python and Deep Learning .   
We've set up an API connection to analyze plant diseases.


## 🪛 How to Start:

1. Clone the Repository
Open your terminal and run the following command to download the project:
git clone [repository-link]
cd Plant-Health-AI  

2. Set Up Virtual Environment (Optional but Recommended)
To keep the dependencies isolated: 

python -m venv venv
# On Windows:
venv\Scripts\activate
# On macOS/Linux:
source venv/bin/activate 

3. Install Dependencies
Install all necessary libraries including Flask, TensorFlow, and CORS:

pip install -r requirements.txt  


   


## 📁 AboutFolders:

src/backend/: Contains the AI analysis engine (api_service.py) and security layers.

src/frontend/: Contains the Web UI (HTML, CSS, JS) developed for the client side. 

main.py: The Flask API Server that bridges the AI model with the Web Interface.  

uploads/: Temporary storage for images processed through the API. 

## ⚙️ How to Run 
    
 start the application: 
    python src/main.py  

 The application will be live at http://127.0.0.1:5000.
